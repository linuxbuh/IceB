                 Ввод и корректировка справочника окладов

 Этот режим позволяет вводить и корректировать оклады по датам их изменения.
Эта информация используется для автоматизированного расчёта начислений и для
автоматизированного расчёта отпускных.
 После входа в этот режим оператор увидит список должостных окладов или
сообщение об отсутствии таковых. В нижней части экрана находится строка
помощи:

F1 помощь F2/+ запись Sh+F2-коррект. F3 удалить F4 поиск F5 расп. F10 выход

F1       - получение подсказки;
F2       - ввод новой записи;
+        - ввод новой записи;
Shift+F2 - корректировка выбранной записи;
F3       - удаление выбранной записи;
F4       - поиск записей по заданным реквизитам поиска;
           символов в наименовании;
F5       - распечатать справочник;
F6       - переключение режимов просмотра
           первое нажатие включение режима сортировки по датам
           второе нажатие включение режима только последние 
           записи изменения оклада
           третье нажатие возврат в режим сортировки по табельным
           номерам
F7       - переключение режимов просмотра
           первое нажатие только работающие
           второе нажатие только уволенные
           третье нажатие возврат в первоначальный режим просмотра
F10      - выход.

 Реквизит "Счёт" может быть не заполнен. В этом случае при
автоматизированном расчёте счёт, который введён в начисление не будет
менятся. Если реквизит "Счёт" заполнен, то этот счёт будет введён в
начисление при автоматизированном расчёте. Если вместо счёта ввести "*", то
в начисление при автоматизированном расчёте будет записан счёт взятый из
карточки работника.

 Если при расчёте отпускных оператору будет необходимо чтобы некоторые
оклады не брались в расчёт, то для этого в реквизит "Коментарий" нужно
ввести символ "-".
