         Установка разрешения оператору для работы с базой

 Для того чтобы оператор имел возможность работать с базой данных ему нужно
установить соответствующие полномочия. 
 После входа в этот режим на экране появиться меню:

---------------------------
|Логин оператора...       |
|Пароль оператора..       |
|Имя базы..........       |
|Хост оператора....       |
|Доступ...(r,w)....       |
---------------------------

F1 помощь F2 установить F4 проверить F10 выход

F1  -получить подсказку
F2  -установить разрешение на работу с базой
F4  -проверить есть ли уже доступ оператору с таким логином 
F10 -выход

   После нажатия клавиши F2 программа проверит есть ли этот логин
в списке операторов указанной базы данных. Если нет, то на экране появиться запрос на ввод
фамилии логина. Программа автоматически запишет этот логин в список
логинов в базу для которой был разрешён доступ, при этом логину будут установлены полномочия
разрешающие ему всё. Для ограничения полномочий логину нужно зайти в промотр списка баз,
выбрать нужную базу и нажать клавишу F6. 
   Программа позволяет как ввести нового пользователя с нужными полномочиями,
  так и перустановить пароль и права доступа для уже введенного
  пользователя.
    В реквизит "Хост пользователя" вводиться хост с которого пользователю
  будет разрешен доступ к базе данных. База данных MySQL позволяет ввести
  любое количество хостов, с которых пользователь может получить доступ к базе
  данных.
    Если вы вводите нового пользователя и хост пользователя не будет введен,
  то пользователь получит доступ к базе с любого хоста.
    При переустановке пароля или прав доступа должен быть обязательно правильно
  введен хост, с которого пользователю уже разрешен доступ, иначе полномочия
  не будут переустановлены.
