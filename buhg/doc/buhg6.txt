                Меню "Работа с файлами распечаток"

        Если оператор заказал, чтобы программа для него сделала
какой-нибудь отчет, то этот отчет будет выгружен в файл имя (имена)
которого программа сообщит оператору, и на экране появится
меню со списком файлов.

        Это меню позволяет просматривать файлы, распечатывать их
на принтере, редактировать. После выхода из этого режима, файл (файлы)
будет уничтожен. Вы можете с помощью клавиши F9 отметить файлы, которые не 
нужно удалять после выхода из этого меню.
Вы можете продолжить с ним работу воспользовавшись режимом "Работа
с файлами распечаток", который находится в главном меню
программы.

В нижней части экрана строка подсказки:

F1       - получение этой подсказки;
F2       - просмотр выбранного файла;
Shift+F2 - просмотр выбранного файла перед печатью;
F3       - редактирование файла;
Shift+F3 - назначить другой редактор файлов;
F4       - получение меню для выбора копирования выбранного файла;
F5       - получение меню для выбора режима - распечатки выбранного файла или см
F6       - просмотр очереди на печать, удаление с очереди на печать;
F7       - включить/выключить просмотр настроек;
F8       - перекодировать выбранный файл в кодировку WINDOWS-1251,
           повторное нажатие клавиши возвращает предыдущую кодировку;
F9       - выставить/снять метку сохранения файла после выхода по F10;
F10      - выход с удалением всех отчётов;
Tab      - поменять местами комментарий и имя файла;
Enter    - просмотр выбранного файла.
