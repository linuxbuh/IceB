                 Загрузка списка контрагентов

	Этот режим применяется для первоначальной загрузки данных в
справочник контрагентов. Данные будут загружаться из файла имя
которого введет оператор.
	После входа в этот режим на экране появится меню с предложением
ввести имя файла в котором находится список контрагентов, который нужно
загрузить. В нижней части экрана строка подсказки:

F1помощь F2/+ загрузка F3 просмотр F10 выход

F1  - получение описания работы режима;
F2  - начать загрузку данных;
+   - начать загрузку данных;
F3  - просмотреть файл имя которого оператор ввел в меню; 
F10 - выход.

	Формат записи файла должен иметь следующий вид:

123|Фирма "Крона"|ПИБ|Винница Порика 5/2|600-летия 4|12345|1234|34567|567889|3456787|22-22-22|1|233456|Договор|Фирма "Крона"|Винница|0
 |       |         |     |                    |        |     |    |      |      |       |     |   |       |        |           |     |
 |       |         |     |                    |        |     |    |      |      |       |     |   |       |        |           |     -> 0-общая система налогообложения 1-единый налог
 |       |         |     |                    |        |     |    |      |      |       |     |   |       |        |           ->Населённый пункт где находится контрагент
 |       |         |     |                    |        |     |    |      |      |       |     |   |       |        ->Полное наименование контрагента (не обязательный реквизит)
 |       |         |     |                    |        |     |    |      |      |       |     |   |       ->Договор (не обязательный реквизит)
 |       |         |     |                    |        |     |    |      |      |       |     |    ->Регистрационный номер частного предпринимателя(не обязательный реквизит)
 |       |         |     |                    |        |     |    |      |      |       |     ->Код группы (не обязательный реквизит)
 |       |         |     |                    |        |     |    |      |      |       ->Телефон (не обязательный реквизит)
 |       |         |     |                    |        |     |    |      |      ->Номер свидетельства плательщика НДС(не обязательный реквизит)
 |       |         |     |                    |        |     |    |      ->индивидуальный налоговый номер(не обязательный реквизит)
 |       |         |     |                    |        |     |    ->Номер счёта в банке (не обязательный реквизит)
 |       |         |     |                    |        |     ->МФО (не обязательный реквизит)
 |       |         |     |                    |        ->Код ЕДРПУО (не обязательный реквизит)
 |       |         |     |                    ->Адрес банка (не обязательный реквизит)
 |       |         |     ->Адрес организации (не обязательный реквизит)
 |       |         ->Наименование банка (не обязательный реквизит)
 |       ->Короткое наименование организации (обязательный реквизит)
 ->Код контрагента (не обязательный реквизит)


 Обязательным реквизитом является только второе поле "Короткое наименование
организации. Если первый реквизит "Код контрагента" будет введён, то
программа проверит свободен этот код или нет. Если на этот код в списке
контрагентов уже введена организация, то программа присвоит этому
контрагенту другой свободный код.
