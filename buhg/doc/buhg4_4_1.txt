             Импорт стартового сальдо по счетам

 Этот режим позволяет импортировать стартовые сальдо по счетам из файла.
В одном файле могут быть сальдо как для счетов с развёрнутым сальдо, так
и для счетов со свёрнутым сальдо, как для балансовых счетов, так и для
небалансовых счетов.

После входа в этот режим на экране появиться


-------------------
|Введите имя файла|
|                 |
-------------------


F1 помощь F2 загруз F3 просмотр F5 настройка F6 выполнить F10 выход

F1  - показ на экран этого текста;
F2  - начать импорт сальдо;
F3  - просмотр файла имя которого ввели в меню;
F4  - просмотр файла введённого в меню;
F5  - настройка команд которые необходимо выполнить до начала импорта;
F6  - выполнить команды в файле настройки (F5);
F10 - выход.

 Если в командах файла настройки должно использоваться имя файла, то 
вместо имени файла нужно ввести ICEB_ZAMENA
Программа перед выполнением команд заменит эту цепочку символов на имя
файла.



  Формат записи данных в файле должен быть такой:

2006|311|10.0|0.|
2006|361|0.|256.12|475|Завод металлоконструкций
 |    |  |    |     |           |
 |    |  |    |     |           -> Краткое наименование контрагента (обязательный реквизит для счетов с развёрнутым сальдо)
 |    |  |    |     -> Рекомендованный код контрагента для счетов с развёрнутым сальдо (необязательный реквизит)
 |    |  |    |      
 |    |  |    -> Сумма по кредиту
 |    |  -> Сумма по дебету
 |    -> Счет
 -> Год


 В процессе загрузки программа будет проверять есть ли счёт в плане счетов.
Если счёта нет запись загружена не будет. 

Для счетов с развёрнутым сальдо необходимо наименование контрагента. 

Если контрагента с указанным наименованием нет в списке контрагентов,
то программа добавит в список контрагентов наименование контрагента 
под рекомендованным кодом контрагента указанным в строке, если этот код не занят.
Если рекомендованный код контрагента занят, или в строке не указан код контрагента, то
будет присвоен уникальный код.
