            Ввод и корректировка списка договоров

	Этот режим позволяет ввести и корректировать список
договоров заключённых с контрагентом. При вводе документов программа
автоматически будет брать реквизиты договора по самой последней дате
если не назначен договор по умолчанию.
	После входа в этот режим на экране появиться список
введённых договоров или сообщение об отсутствии записей в списке
договоров. В нижней части экрана появиться строка подсказки:

F1 помощь F2/+ запись F3 удалить F4 поиск F5 печать F10 выход

F1       - получение подсказки;
F2       - ввести новую запись в список суббалансов;
Shift+F2 - корректировать запись на которой установлена
           подсветка строки;
F3       - удалить запись на которой установлена подсветка
           строки;
F5       - распечатать список договоров;
F6       - установить/снять метку договора по умолчанию;
F10      - выход.
