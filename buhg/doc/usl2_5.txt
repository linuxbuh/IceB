                    Импорт документов

	Этот режим позволяет импортировать документы из ASCII
файла в базу данных для подсистемы "Учёт услуг".

В ниженй части экрана располагается строка подсказки:


F1 помощь F2 загрузить F3 просмотр F4 очистить F5 настройка F6 выполнить F10 выход

F1  - получить на экран этот текст;
F2  - начать импорт проводок из указанного в меню файла; 
F3  - просмотреть на экране файл указанный в меню;
F4  - очистить строку ввода;
F5  - настроить список команд которые необходимо выполнить до начала импорта;
F6  - выполнить список введённых команд (F5);
F10 - выход.

 Если в командах файла настройки должно использоваться имя файла, то 
вместо имени файла нужно ввести ICEB_ZAMENA
Программа перед выполнением команд заменит эту цепочку символов на имя
файла.

                 Описание формата записи

 В начале каждой строки должен быть код записи. Программа
понимает следующие коды записи:

KON  - Реквизиты контрагента (не обязательная запись)
DOCU - Шапка документа
ZPU  - Запись в документе

	В одном файле может быть любое количество документов.
Первой строкой должна быть строка "DOCU", все строки до следующей 
строки "DOCU" считаются принадлежащими этому документу.
Реквизиты в строке отделяются друг от друга символом "|".

Формат записи шапки документа имеет следующий вид:

DOСU|+|1.1.2004|1|494|1+34|1234|ПП|0|Госстандарт|   |   |    |    |     |
 |   |    |     |  |   |    |   |  |    |         |   |   |    |     |
 |   |    |     |  |   |    |   |  |    |         |   |   |    |     ->дата доверенности
 |   |    |     |  |   |    |   |  |    |         |   |   |    ->доверенность
 |   |    |     |  |   |    |   |  |    |         |   |   ->через кого
 |   |    |     |  |   |    |   |  |    |         |   -> условие продажи
 |   |    |     |  |   |    |   |  |    |         -> основание
 |   |    |     |  |   |    |   |  |    -> Наименование контрагента
 |   |    |     |  |   |    |   |  -> НДС 0-20% 1-0% на територии Украины 2-0% экспорт 3-0% освобождение
 |   |    |     |  |   |    |   -> Код операции
 |   |    |     |  |   |    ->Номер налоговой накладной
 |   |    |     |  |   -> Номер документа (если поле не заполнено программа сама присвоит номер документа)
 |   |    |     |  -> Рекомендованный код контрагента (не обязательное поле для заполнения)
 |   |    |     -> Код подразделения
 |   |    -> Дата документа
 |   -> "+" приходный документ  "-" расходный документ
 -> Код строки




Формат записи в документе имеет следующий вид:

ZPU|134|234.3|34.0|Кв|232|за 10.2005 г.|Электроэнергия
 |   |    |     |  |   |       |            |
 |   |    |     |  |   |       |            -> Наименование услуги
 |   |    |     |  |   |       -> дополнение к наименованию услуги (необязательный реквизит)
 |   |    |     |  |   -> Счёт учета
 |   |    |     |  -> Единица измерения
 |   |    |     -> Цена
 |   |    -> Количество
 |   -> Код услуги (не обязательный реквизит если введено наименование услуги)
 -> Код строки
